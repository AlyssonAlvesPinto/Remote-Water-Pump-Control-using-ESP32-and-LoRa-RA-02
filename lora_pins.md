### LoRa RA-02 Pin Mapping with ESP32

| Function | LoRa RA-02 | ESP32   |
|----------|------------|---------|
| MISO     | MISO       | GPIO19  |
| MOSI     | MOSI       | GPIO23  |
| SCK      | SCK        | GPIO18  |
| NSS      | SS         | GPIO05  |
| RST      | RESET      | GPIO14  |
| DIO0     | DIO0       | GPIO02  |
